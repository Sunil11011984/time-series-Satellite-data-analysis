## Basic Execution Steps

1. Ensure that the Data Folder and abc.ui file are at the same location 
2. A sample data set is given in the data folder 
3. Run ```TSSDV.py ```

## Files Included

1. abc.ui
2. Documentation.pdf
3. Readme.txt
4. TSSDV.py
5. Data(folder)
	1. globe.ico
	2. clip(folder)
		* samples of tif
	3. ShapeFile(folder)
		* sample of shapefile
